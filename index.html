Noticify
Noticify is a dynamic web application developed using the MERN stack, designed to keep users updated with the latest notices from various organizations.
Key Features:
  • User Registration and Login: Users can create an account and log in to access the portal.
  • Subscription to Organizations: Users have the ability to subscribe to different organizations.
  • Admin Notice Management: Administrators have the ability to perform CRUD (Create, Read, Update, Delete) operations on the notices within their organization.
